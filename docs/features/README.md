# 追加機能一覧

MVP完了後に追加できる機能の一覧です。
順序は問わず、必要に応じて実装してください。

---

## 基本機能強化

| 機能 | 説明 | 優先度 |
|------|------|--------|
| [ファイル変換](./file-conversion.md) | 音声/動画ファイルをWAVに変換 | 推奨 |
| [エクスポート](./export.md) | TXT/SRT/VTT形式で結果を保存 | 推奨 |
| [エラーハンドリング](./error-handling.md) | 構造化されたエラー表示 | 推奨 |
| [設定永続化](./settings.md) | アプリ設定の保存・復元 | 推奨 |

---

## UX改善

| 機能 | 説明 | 優先度 |
|------|------|--------|
| [多言語対応](./i18n.md) | 日本語/英語の切り替え | 任意 |
| [トースト通知](./toast.md) | 操作結果のフィードバック | 任意 |
| [キーボードショートカット](./keyboard-shortcuts.md) | パワーユーザー向け操作 | 任意 |
| [アニメーション](./animations.md) | UIの視覚的強化 | 任意 |

---

## 高度な機能

| 機能 | 説明 | 優先度 |
|------|------|--------|
| [リアルタイム録音](./recording.md) | マイクから直接文字起こし | 任意 |

---

## 優先度について

- **推奨**: 実用的なアプリとして使うために推奨
- **任意**: ユーザー体験向上のためのオプション
