# 追加機能一覧

MVP完了後に追加できる機能の一覧です。
順序は問わず、必要に応じて実装してください。

---

## 基本機能強化

| 機能 | 説明 | 優先度 |
|------|------|--------|
| [ダッシュボード](./dashboard.md) | 初期画面・クイックアクション・状態一覧 | 推奨 |
| [ファイル変換](./file-conversion.md) | 音声/動画ファイルをWAVに変換 | 推奨 |
| [エクスポート](./export.md) | TXT/SRT/VTT形式で結果を保存 | 推奨 |
| [エラーハンドリング](./error-handling.md) | 構造化されたエラー表示 | 推奨 |
| [設定永続化](./settings.md) | アプリ設定の保存・復元 | 推奨 |
| [履歴管理](./history.md) | 文字起こし履歴の保存・検索・削除 | 推奨 |

## 優先度について

- **推奨**: 実用的なアプリとして使うために推奨
- **任意**: ユーザー体験向上のためのオプション
